・vue.js

Vueオブジェクトは次の3の要素を持っている
el <- どのDOM要素と結びつくか
data <- データ
method <- メソッド

Vue.jsではjs側でデータとメソッドを定義して置くことで、html上でjs的な操作ができるようになる


Vue.jsの基本

MVVM
[View]
<div id="my_view">
</div>

[Model]
var myModel = {
  name: "Ashley",
  age: 24
};

[View-Model] <- Vue
var myViewModel = new Vue({
  el: '#my_view',
  data: myModel
});
